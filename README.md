# brawl_shop
Telegram-бот для небольшого бравл-шопа: каталог товаров, оформление заказа и отправка данных для оплаты. Вы выбираете позицию, указываете контакты и прикрепляете скрин платежа. Администратор управляет категориями и товарами, принимает/отклоняет заявки и ведет историю покупок. Заказы хранятся в локальной SQLite.
