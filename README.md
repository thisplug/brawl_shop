# Brawl Shop Bot

Telegram-бот для небольшого бравл-шопа: каталог товаров, оформление заказа и отправка данных для оплаты. Вы выбираете позицию, указываете контакты и прикрепляете скрин платежа. Администратор управляет категориями и товарами, принимает/отклоняет заявки и ведет историю покупок. Заказы хранятся в локальной SQLite.

## Установка

1. Установите Python 3.10+.
2. Создайте виртуальное окружение (рекомендуется) и активируйте его.
3. Установите зависимости:
   - `pip install -r requirements.txt`

## Настройка

1. Скопируйте файл `.env.example` в `.env`.
2. Заполните переменные:
   - `BOT_TOKEN` — токен бота от `@BotFather`
   - `ADMIN_IDS` — ID администраторов через запятую
   - `DATABASE_PATH` — путь к базе данных (например, `brawl_shop.db`)

> `.`env` нельзя коммитить в GitHub — используйте `.env.example`.

## Запуск

- `python main.py`

## Важно

- База данных `brawl_shop.db` создается/обновляется локально и не предназначена для публикации.
- Кэш Python (`__pycache__`) и служебные файлы игнорируются через `.gitignore`.
